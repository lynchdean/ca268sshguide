CA268 Toodle Remote Access

View on GitHub Download .zip Download .tar.gz

PuTTY set up:

  1. Download PuTTY
  2. Run putty.exe
  3. In the 'Category' section, select Connection>SSH>Tunnels
    • Type "8080" in the source port
    • Underneath, select Dynamic and Auto
    • Click "Add"
